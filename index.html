Good Morning Computer

Good Morning Computer is a friendly morning meetup for computer or computer-curious people in Baltimore. We drink coffee (or not), talk about and/or write code. THIS EVENT IS FOR ANYONE.

No sponsors, no speakers, no particular agenda.
